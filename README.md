# The Sorting Hat Report  
*A Daily Prophet–style productivity reflection tool*

> “In a thousand years of sorting, I have never been fooled.”

The Sorting Hat Report is a Chrome extension paired with a local Node.js daemon that analyzes your daily development activity and browsing behavior to generate a whimsical, AI-crafted efficiency report inspired by the Daily Prophet.

Instead of relying solely on dashboards and numbers, this project reframes productivity reflection as a playful narrative experience.

---

## Overview

This system observes how your day unfolded across:

- Coding activity  
- Git commits  
- Terminal usage  
- Browser tabs and page titles  

It then synthesizes these signals using AI to produce a stylized newspaper report evaluating:

- Focus vs distraction  
- Alignment between intentions and actions  
- Coding effort and output  
- Workflow patterns  

The result is humorous, visually engaging, and surprisingly insightful.

---

## Core Idea

Traditional productivity tools emphasize metrics and charts.

**What if self-evaluation felt like reading a magical newspaper written about your day?**

By combining behavioral signals with AI-generated narrative feedback, The Sorting Hat Report transforms raw activity logs into:

- A Daily Prophet front page  
- O.W.L.–style grades  
- A Delulu Score™  
- Character commentary  
- Quidditch efficiency charts  

Reflection becomes entertaining rather than mechanical.

---

## Features

### Daily Prophet Report  
A newspaper-style layout summarizing and evaluating your day.

### Delulu Score™  
A 0–100 score estimating the gap between perceived productivity and observed behavior.

### O.W.L. Grades  
Category-based performance evaluation:

- Coding  
- Terminal Usage  
- Learning  
- Focus  
- Intentions  
- Overall  

### Character Commentary  
AI-generated editorial remarks delivered in themed voices.

### Quidditch Efficiency Chart  
Visualization of productivity trends across 7 or 30 days.

### Site Classification  
Each visited page is analyzed relative to your stated intentions.

### Hat Speech  
A final AI verdict summarizing your day.
